What is Python

Python is a high-level, general-purpose programming language for a variety of industries. Its language constructs and object-oriented approach are designed to help programmers write clear, logical code for small and large projects. Python can run on almost all platforms, Guido van Rossum began working on Python in the late 1980s as a successor to the ABC programming language, and first released Python 0.9.0 in 1991.
